Este proyecto es una tienda demo que hice para la escuela. Aquí se pueden ver productos, filtrarlos por categorías y también agregarlos al carrito. La idea es que funcione como una tienda básica pero que se vea ordenada y fácil de usar.

Qué incluye el proyecto

Una página principal con los productos.

Cuatro filtros: Electrónica, Ropa, Libros y Todos.

Un carrito que guarda los productos que se agregan.

Un diseño con una paleta de colores personalizada.

El funcionamiento está hecho con HTML, CSS y JavaScript.

Archivos del proyecto

index.html → estructura de la página.

style.css → estilos y colores.

script.js → lógica de los filtros, productos y carrito.

Notas

Es un proyecto de demostración, no es una tienda real.

Los productos son solo ejemplos.

Si algo no funciona, normalmente es porque el CSS o el JS no están bien enlazados.
